> Typed version of can-use-dom to check if you can use the DOM.

## Installation

```sh
yarn add @bconnorwhite/can-use-dom
```

```sh
npm install @bconnorwhite/can-use-dom
```

## API

```js
import canUseDOM from "@bconnorwhite/can-use-dom";

// canUseDOM: boolean
```
